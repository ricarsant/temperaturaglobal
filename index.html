GLOBAL Land-Ocean Temperature

Calentamiento global: Perido 1880 - 2018

Index in 0.01 degrees Celsius

Funte Datos: https://data.giss.nasa.gov/gistemp/

Abstracción WHAT- WHY – HOW :

De acuerdo con Framework de Tamara Munzner:

 

WHAT:

Ø  Data and Dataset Types:

è Tables – Items – Atributes:

o   Data Set con 139 Ítems y 13 Atributos.

o   Atributos usados en la Visualización:

o   Año:  Ordenado– Secuencia – Cuantitativo

o   12 atributos: Correspondiente a las temperaturas por meses del año.: Ordenado – Divérgete -Cuantitativo.

 

Ø  Cada registro corresponde a un año y a las temperaturas registradas por mes en ese año.

 

 

 

WHY:

Ø  Tarea Principal:

Mostrar el crecimiento sistemático en la temperatura global del planeta año a año.

 

De acuerdo con el framework de Tamara, la visualización logra el objetivo mediante:

 

o   Present -> Trends.

o   Search -> Locate -> Outliers.

o   Query -> Summarize -> Features.

o   Query - > Identify -> Similarity.

 

Insght: El calentamiento global es evidente y es necesario tomar conciencia para detener esta situación.

 

 

Ø  Tarea Secundaria:

Mostrar que a pesar de a ver decrementos en la temperatura en ciertos periodos, la tendencia agregada es en alza.

 

Insght: Se ve que, para los últimos años, y en especial para 2018, se evidencian temperatura a la baja. Como resultado de todas las medidas adoptadas a nivel global.

 

 

De acuerdo con el framework de Tamara, la visualización logra el objetivo mediante:

 

o   Discover -> Outliers.

o   Serach -> Explore -> Extremes.
